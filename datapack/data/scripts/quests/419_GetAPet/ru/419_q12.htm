Вопрос:
Хозяин может использовать своего питомца для различных целей. Для каких целей НЕ может использоваться питомец?
Для перевозки вещей.
Помогать в бою и охоте, нападая на цель.
Сбегать в деревню и купить какие-нибудь вещи.
Если нужны деньги, то питомца можно продать.
Может использоваться для любой из этих целей.
