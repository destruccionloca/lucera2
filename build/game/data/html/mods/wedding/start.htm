Приветствую вас молодожёны! Вы хотели обручиться? Брак является дорогостоящей церемонией! Вы уверены, что хотите продолжить?
Пожениться

Для того чтобы пожениться:
Сначала вам необходимо помолвиться. Чтобы помолвиться нужно выбрать партнера и ввести команду .engage. При этом вы должны быть у друг друга в списке друзей. Для того чтобы добавить в друзья нужно ввести /friendinvite PlayerName где PlayerName это имя вашего партнера.

После того как помолвитесь, вы оба должны одеть Свадебный Наряд необходимый для свадьбы. Свадьба вам обойдется в %price% Аден.

Как только вы будете обручены, можно использовать команду .gotolove для телепорта к вашему партнеру в любое время. Если вы хотите развестись, просто наберите команду .divorce.
